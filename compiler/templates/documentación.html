Matarife.jS

Matarife es un lenguaje de programación creado por Mateo gomez en el backend, Diogenes Arrieta en el frontend y Yesid rodriguez como desarrollador y testing. Como proyecto de ultimo seguimiento en la asignatura de Compiladores e Intérpretes, impartida por el Docente e Ingeniero de sistemas Daniel González Polo, en la Universidad del Magdalena. El lenguaje está basado en una gramática muy sencilla y de fácil escritura, a continuación, veremos aspectos importantes de este:

1. Tipos de datos 1.1. Entero (E), Flotante (F), String (S) 1.2. La forma de declarar las variables en Matarife.jS es: (E) número (F) Flotante (S) cadena

1.3. Operador de asignación El operador de asignación es el (=), como en la mayoría de los lenguajes de programación existentes.

3. Sentencias 3.1. Sentencia de control si/sino/sino si. si (expresión lógica) { !instrucciones! } sino { !instrucciones! }"

4. Bucles 4.1. Bucle para/H/MQ Para: P (i=n; i<número; i++) { !instrucciones! } H (Hacer): H{ !instrucciones! }MQ( expresión lógica ); MQ (Mientras Que): MQ (expresión lógica) { !instrucciones! }
